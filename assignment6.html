Welcome to my Multiplication Table Generator

Directons: Enter the start and end number of both the horizonal and vertical axis of the table (labels on form). A table will be generated below the form where every cell in the table contains the product of the first horizontal cell and the first vertical cell.

Restrictions:
Each number can be between 9,007,199,254,740,991 and -9,007,199,254,740,991.
Horizontal and Vertical Ranges must be within 1000.

Example: Start Number (Horizontal): 1249, End Number (Horizontal): 300 (1249 - 300 = 949)
