Welcome to my Multiplication Table Generator V3.0 (with tabs and sliders)

Directons: Enter the start and end number of both the horizonal and vertical axis of the table (labels on form). A table will automatically be generated below the form where every cell in the table contains the product of the first horizontal cell and the first vertical cell.

Restrictions:
Each number can be between -1000 and 1000.
You can also use the silders to change your values
Horizontal and Vertical Ranges must be within 1000.

Example: Start Number (Horizontal): 1000, End Number (Horizontal): 125 (1000 - 125 = 875)

Horizontal Range is greater than 1000
Vertical Range is greater than 1000
